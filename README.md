# Super-Store-Sales-Profit-Analysis-Dashboard  

As part of my data analytics learning journey, I worked on a real-world retail sales dataset to analyze business performance and generate meaningful insights using data visualization techniques.

🔧 Tools & Skills Used: ✔️ Power BI
✔️ SQL
✔️ Data Analysis
✔️ Data Visualization
✔️ Dashboard Design

📊 Key Insights from the Dashboard: ✅ Analyzed overall sales of 1.57M+ and profit of 175K+
✅ Identified top-performing categories like Office Supplies and Technology
✅ Analyzed customer payment preferences and shipping modes
✅ Tracked monthly sales and profit trends
✅ Created interactive regional sales analysis using map visuals
